## Summary
Databricks has announced that External Lineage (formerly Bring Your Own Lineage) in Unity Catalog is now generally available. The feature allows users to register data assets that exist outside of Databricks—such as Salesforce, MySQL, Tableau, and Power BI—so that a unified lineage graph can span the entire data flow. As part of this release, Lakeflow Connect managed ingestion pipelines now automatically record source lineage from source tables to their destination tables in Unity Catalog.

## Key claims
- External Lineage in Unity Catalog is now generally available, having previously been called Bring Your Own Lineage.
- Users can register external assets such as Salesforce, MySQL, Tableau, and Power BI into Unity Catalog's lineage graph.
- The feature enables a single lineage graph that spans data flows both inside and outside Databricks.
- Lakeflow Connect managed ingestion pipelines now automatically record source lineage from source tables to destination tables in Unity Catalog.
