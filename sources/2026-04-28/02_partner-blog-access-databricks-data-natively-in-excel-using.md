## Summary
Databricks has released a new Excel Add-in, currently in Public Preview, that allows users to directly access data from their Databricks Lakehouse within Microsoft Excel. This add-in eliminates the need for manual data exports or complex integrations, enabling business stakeholders to work with live data. Users can browse Unity Catalog tables, run SQL queries, and import data directly into spreadsheets without leaving Excel, requiring no SQL knowledge for basic operations. The add-in supports Excel for the Web, Windows, and macOS, and uses Single Sign-On (SSO) for authentication.

## Key claims
- Databricks has launched a new Excel Add-in for native data access.
- The add-in connects Databricks workspaces directly to Microsoft Excel.
- Users can browse Unity Catalog tables and run SQL queries from within Excel.
- Live Lakehouse data can be pulled into spreadsheets without leaving Excel.
- The add-in supports basic table imports without requiring SQL knowledge.
- Full custom query support is available for advanced users.
- The add-in uses Single Sign-On (SSO) for authentication and does not support Personal Access Tokens (PATs).
- The add-in is available for Excel for the Web, Excel Desktop on Windows, and Excel Desktop on macOS.
