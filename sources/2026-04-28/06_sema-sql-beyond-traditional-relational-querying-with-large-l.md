## Summary
SEMA-SQL is a new system designed to bridge the gap between traditional relational databases and the advanced reasoning capabilities of Large Language Models (LLMs). It addresses the limitations of standard SQL and existing text-to-SQL systems by enabling semantic querying and information extraction beyond explicit schemas. SEMA-SQL automates the generation, optimization, and execution of queries that combine relational operations with LLM-powered functions, significantly reducing LLM invocations.

## Key claims
- Real-world queries often require capabilities beyond standard SQL, including semantic entity matching and analysis of unstructured text.
- Existing text-to-SQL systems are limited to relational operations and cannot fully leverage LLM semantic reasoning.
- SEMA-SQL automatically generates efficient queries combining relational operations with LLM semantic reasoning.
- Hybrid Relational Algebra (HRA) is introduced as a declarative abstraction unifying traditional relational operators with LLM user-defined functions (UDFs).
- SEMA-SQL automates query generation using in-context learning for precise natural language specifications of LLM UDFs.
- The system optimizes queries through cost-based transformations and UDF rewriting.
- SEMA-SQL reduces LLM invocations by an average of 93% in semantic joins through intelligent batching.
