## Summary
Databricks has announced that Lakeflow Spark Declarative Pipelines parameters are now in Beta on AWS. This feature allows users to define key-value pairs at the pipeline level and reference them from SQL source code using named parameter syntax. The release note points users to documentation on how to use parameters with pipelines.

## Key claims
- Lakeflow Spark Declarative Pipelines parameters are now in Beta on Databricks AWS.
- Pipeline parameters allow users to define key-value pairs at the pipeline level.
- Parameters can be referenced from SQL source code using named parameter syntax.
