## Summary
Databricks has announced that automatic identity management with Microsoft Entra ID is now generally available (GA). This feature allows organizations to synchronize users, service principals, and groups from Microsoft Entra ID to Databricks without requiring SCIM provisioning. Once enabled, administrators can search for identities in identity-federated workspaces and add them directly, with group memberships remaining continuously in sync with Microsoft Entra ID.

## Key claims
- Automatic identity management with Microsoft Entra ID has reached general availability on Databricks.
- The feature enables syncing of users, service principals, and groups from Microsoft Entra ID to Databricks without SCIM provisioning.
- When enabled, users can search for identities in identity-federated workspaces and add them directly to a workspace.
- Group memberships are kept automatically in sync with Microsoft Entra ID.
- The feature is available on Databricks on AWS as of May 2026.
