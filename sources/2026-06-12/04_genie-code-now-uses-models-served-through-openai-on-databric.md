## Summary
Databricks has updated its Genie Code feature to support models served through OpenAI on Databricks, expanding the existing support for Azure OpenAI and Anthropic on Databricks. This change applies when partner-powered AI features are enabled on the platform. Users can consult the Partner-powered AI features documentation to see which model provider is used for each specific feature.

## Key claims
- Genie Code now uses models served through OpenAI on Databricks.
- Prior to this update, Genie Code supported models served through Azure OpenAI and Anthropic on Databricks.
- The expanded model support is available when partner-powered AI features are enabled.
- Databricks provides documentation listing the model provider used by each partner-powered AI feature.
