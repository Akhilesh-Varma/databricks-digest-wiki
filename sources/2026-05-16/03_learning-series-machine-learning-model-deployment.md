## Summary
Databricks Academy offers a free course called Machine Learning Model Deployment, the third installment in the Machine Learning with Databricks series. The course teaches ML practitioners how to move models from development into production using Databricks tools and workflows. It covers batch, pipeline, and real-time deployment strategies, including use of Delta Live Tables, Model Serving, and MLflow. The course is approximately two hours long, structured into four sections with nineteen lessons, and targets associate-level learners familiar with Python, Spark, and MLflow.

## Key claims
- Databricks Academy offers the Machine Learning Model Deployment course for free.
- The course is the third in the Machine Learning with Databricks series.
- It covers three deployment strategies: batch, pipeline, and real-time inference.
- Delta Live Tables (DLT) is highlighted as a tool for scheduled and pipeline-based model deployment.
- Model Serving and serving endpoints are presented as the solution for low-latency real-time inference.
- MLflow deployment features are evaluated as part of the course curriculum.
- The course consists of 4 sections and 19 lessons with a duration of approximately 2 hours.
