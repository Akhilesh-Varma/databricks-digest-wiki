## Summary
Codiff is a local diff review tool built by the author to address limitations felt when reviewing large amounts of LLM-generated code. Inspired by sites like diffs.com and trees.software, the author used an LLM to scaffold the initial app in 16 minutes, then polished it with additional features. Codiff offers file filters, search, an LLM walkthrough mode, and review comments that can be pasted back into an LLM. It is designed to be fast, minimal, and beautiful for handling large diffs locally.

## Key claims
- Codiff was built as a local diff review tool to replace git delta for reviewing LLM-generated code.
- The initial version of Codiff was built in 16 minutes by pointing an LLM at diffs.com and trees.software as references.
- Codiff supports file filters, search, an LLM walkthrough mode, and copy-pasteable review comments.
- The tool is described as extremely fast for large diffs.
- The author released Codiff as an open-source project encouraging forks.
