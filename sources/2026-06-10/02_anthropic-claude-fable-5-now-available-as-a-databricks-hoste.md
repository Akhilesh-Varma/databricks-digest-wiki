## Summary
Databricks Model Serving has added support for Anthropic Claude Fable 5 as a Databricks-hosted model. Claude Fable 5 is designed for autonomous knowledge work and coding, capable of handling long-running, complex, and asynchronous tasks with reduced human intervention. Users can access the model through Foundation Model APIs on a pay-per-token basis.

## Key claims
- Databricks Model Serving now supports Anthropic Claude Fable 5 as a hosted model.
- Claude Fable 5 is built for autonomous knowledge work and coding tasks.
- The model is designed to handle long-running, complex, and asynchronous tasks.
- Claude Fable 5 requires less human intervention compared to prior models.
- Access to Claude Fable 5 on Databricks is available via Foundation Model APIs with pay-per-token pricing.
