## Summary
sqlit is an open-source terminal user interface (TUI) for SQL databases, inspired by the lazygit workflow. It supports a wide range of databases including PostgreSQL, MySQL, SQLite, DuckDB, Snowflake, BigQuery, and many others. The tool is built with Python and the Textual framework, and requires no configuration files to get started. It includes features such as Docker integration, SSH tunnels, Vim-style editing, fuzzy filtering, cloud CLI integration, and OS keyring credential storage.

## Key claims
- sqlit is a lazygit-style TUI for SQL databases that requires no config files or documentation to start using.
- It supports over 25 databases including PostgreSQL, MySQL, SQLite, DuckDB, Snowflake, BigQuery, Databricks, and CockroachDB.
- sqlit is built with Python and the Textual framework.
- It features Docker integration that auto-detects and connects to running database containers.
- The tool supports SSH tunnels, OS-keyring credential storage, and password manager integration.
- Cloud CLI integration allows browsing external databases via Azure, AWS, and GCP CLIs.
- sqlit can be installed via pipx, uv tool install, or pip.
