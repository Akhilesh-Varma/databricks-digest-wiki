## Summary
Databricks has added documentation for five new SQL statements to its SQL reference, all focused on managing governed tags. The new statements are CREATE GOVERNED TAG, ALTER GOVERNED TAG, DROP GOVERNED TAG, DESCRIBE GOVERNED TAG, and SHOW GOVERNED TAGS. These additions allow users to create, modify, delete, describe, and list governed tags directly through SQL. Further details are available in the 'Create and manage governed tags' documentation.

## Key claims
- Databricks SQL reference now includes documentation for five new SQL statements for managing governed tags.
- The five new statements are CREATE GOVERNED TAG, ALTER GOVERNED TAG, DROP GOVERNED TAG, DESCRIBE GOVERNED TAG, and SHOW GOVERNED TAGS.
- These SQL statements enable users to create, modify, delete, describe, and list governed tags.
- The feature was announced as part of Databricks April 2026 release notes.
