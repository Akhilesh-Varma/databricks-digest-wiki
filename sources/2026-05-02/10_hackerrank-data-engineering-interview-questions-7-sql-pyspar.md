## Summary
This article covers seven canonical SQL and PySpark patterns that HackerRank tests in data engineering interviews. The clusters include SQL JOINs with foreign-key filters, multi-table aggregation, GROUP BY HAVING predicates, window functions for streaks and per-day winners, CASE expressions on hierarchies, scalar subqueries with strict output formatting, and PySpark DataFrame transforms. Each section provides a teaching explanation, sub-topic primitives, a worked interview-style problem, a runnable solution, and a step-by-step trace. The guide emphasizes that HackerRank's grader is exact-match on stdout, meaning minor formatting errors like a wrong trailing zero will fail a test case.
