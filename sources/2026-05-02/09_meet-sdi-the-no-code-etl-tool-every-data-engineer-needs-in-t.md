## Summary
Simple Data Integrator (SDI) is a no-code ETL tool designed to eliminate the need for custom glue code when syncing data between databases, cloud storage, and SaaS platforms. It supports native connections to SQL Server, Oracle ADB, PostgreSQL, MySQL, AWS S3, OCI Object Storage, Google Cloud Storage, and Azure Blob. SDI offers bidirectional Salesforce sync without requiring Apex or Python code, and includes a gateway component that enables secure on-premise-to-cloud connectivity without VPN tunnels. Built-in features such as automatic retries, parallel loading, and integrity monitoring handle operational reliability automatically. The tool positions itself as a lightweight alternative to both heavy enterprise ETL platforms and fragile custom scripts.

## Key claims
- SDI enables syncing of SQL Server, Oracle ADB, PostgreSQL, and MySQL databases to cloud storage without custom scripting.
- SDI supports bidirectional Salesforce sync without writing Apex or Python code.
- The SDI Gateway allows on-premise databases to connect to the cloud securely without VPN tunnels.
- SDI includes automatic retries, parallel loading, and integrity monitoring as built-in operational features.
- SDI supports REST, SOAP, and OData APIs for SaaS integration.
- SDI targets cloud storage platforms including AWS S3, OCI Object Storage, Google Cloud Storage, and Azure Blob.
