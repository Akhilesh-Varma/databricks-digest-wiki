## Summary
Databricks has released a Data Intelligence Platform Cheat Sheet, a concise reference document covering the key features and functionalities of the Databricks platform. The sheet is designed for data engineers, analysts, and data scientists, covering areas such as governance with Unity Catalog, generative AI with Mosaic AI, and data engineering with Lakeflow. It is intended for printing, sharing, and use at workshops and events. The cheat sheet will be periodically updated as the platform evolves, with an April 2026 release adding features like Lakewatch, Genie, Code Chat, Lakeflow Designer, and Spark Declarative Pipelines.

## Key claims
- Databricks released a Data Intelligence Platform Cheat Sheet as a single-page reference for platform capabilities.
- The cheat sheet covers governance and security with Unity Catalog, generative AI with Mosaic AI, and data engineering with Lakeflow.
- The document is intended for data engineers, analysts, and data scientists.
- The cheat sheet will be periodically updated to reflect new product capabilities.
- An April 2026 update added Lakewatch, Genie, Code Chat in Databricks, Lakeflow Designer, and Spark Declarative Pipelines.
