## Summary
Databricks has announced the removal of BI compatibility mode from the Power BI connector, which previously enabled querying Unity Catalog Metric Views from external BI tools like Power BI. This change has caused reports relying on this mode to stop functioning. Databricks suggests advocating for restored support on the Fabric Ideas forum and recommends migrating to Databricks AI BI dashboards as a native alternative that seamlessly integrates with Metric Views.

## Key claims
- BI compatibility mode allowed querying Unity Catalog Metric Views from external BI tools like Power BI.
- Databricks has removed the BI compatibility mode option from the Power BI connector.
- Reports that relied on the BI compatibility mode connector option have stopped working.
- Users can advocate for the restoration of Databricks metric views support in Power BI by voting on the Fabric Ideas forum.
- Databricks recommends using Databricks AI BI dashboards as a native alternative to Power BI.
- Databricks AI BI dashboards work natively with metric views, eliminating the need for translation layers.
