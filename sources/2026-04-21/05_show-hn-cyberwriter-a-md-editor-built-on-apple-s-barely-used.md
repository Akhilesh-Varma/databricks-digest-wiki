## Summary
Apple has introduced a comprehensive on-device AI stack in macOS, featuring a 3B-parameter foundation model, a natural language framework, and speech-to-text capabilities. A developer has built CyberWriter, a Markdown editor, on top of these features, allowing users to leverage AI capabilities without requiring API keys, cloud calls, or per-token costs. The app indexes Markdown folders, enables semantic search, and provides features like voice notes and dictation. The developer notes that while the AI capabilities have limitations, the developer experience for these APIs is good.

## Key claims
- Apple has shipped a 3B-parameter foundation model with API access in macOS 26.
- The model allows for streaming, structured output, and tool use without requiring an API key or cloud call.
- CyberWriter, a Markdown editor, has been built on top of Apple's on-device AI stack, enabling users to leverage AI capabilities without requiring API knowledge.
- The app indexes Markdown folders via NLContextualEmbedding and enables semantic search with a 'Related Ideas' section.
- The app provides features like voice notes, dictation, and inline quick actions, all of which run locally on the device.
