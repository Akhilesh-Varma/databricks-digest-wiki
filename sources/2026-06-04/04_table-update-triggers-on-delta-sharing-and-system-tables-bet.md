## Summary
Databricks has released a beta feature that extends table update triggers in Lakeflow Jobs to support Delta Sharing and system tables. Previously limited to local tables, triggers can now fire whenever a Delta Sharing provider updates a shared table or view, or when new operational data arrives in a system table. This capability is currently restricted to Databricks-to-Databricks Delta Sharing scenarios only.

## Key claims
- Table update triggers in Lakeflow Jobs now support Delta Sharing and system tables in addition to local tables.
- Jobs can be triggered whenever a Delta Sharing provider updates a shared table or view.
- Jobs can also be triggered when new operational data lands in a system table.
- This feature is limited to Databricks-to-Databricks Delta Sharing only.
- The feature is currently in beta.
