## Summary
This Databricks Community roundup highlights eight recent technical blog posts covering a range of topics. These include query performance tuning for Databricks Serverless SQL, a DIRECTV customer story using Databricks Genie for conversational analytics, and best practices for safe data experimentation. Additional posts cover Delta Lake MERGE optimization, a metadata-driven ETL framework, and Apache Spark's Real-Time Mode applied to gaming sessionization. A free Databricks Academy course on AI BI for self-service analytics is also featured.

## Key claims
- Databricks Serverless SQL (DBSQL) is positioned as Databricks' latest offering for building data warehouses on the Lakehouse.
- DIRECTV uses Databricks Genie to enable plain-language querying instead of writing SQL for conversational analytics.
- A Delta Lake MERGE operation can be up to 5x slower than a full table overwrite under certain conditions.
- Databricks Academy offers a free course titled 'Databricks AI BI for Self-Service Analytics' targeting business users and analysts.
- Apache Spark's Real-Time Mode is demonstrated through a gaming sessionization use case with a self-serve GitHub repo.
- A metadata-driven ETL framework in Databricks is presented as a solution for managing complex data flows in modern enterprises.
