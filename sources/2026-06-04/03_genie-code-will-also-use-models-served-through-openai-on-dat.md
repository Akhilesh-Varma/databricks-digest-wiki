## Summary
Databricks is expanding Genie Code to support models served through OpenAI on Databricks, in addition to the existing support for Azure OpenAI and Anthropic on Databricks. This change applies when partner-powered AI features are enabled on the platform. The update broadens the range of AI model providers available to Genie Code users.

## Key claims
- Genie Code will soon support models served through OpenAI on Databricks.
- Genie Code already supports models served through Azure OpenAI and Anthropic on Databricks.
- The expanded model support is contingent on partner-powered AI features being enabled.
