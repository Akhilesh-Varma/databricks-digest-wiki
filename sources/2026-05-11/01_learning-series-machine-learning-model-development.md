## Summary
Databricks Academy offers a free course called Machine Learning Model Development, the second in the Machine Learning with Databricks series. The course teaches ML practitioners how to build, tune, and improve traditional machine learning models on the Databricks Data Intelligence Platform. It covers core workflows including regression and clustering, experiment tracking with MLflow, hyperparameter tuning with Optuna, and automated model building with Databricks AutoML. The course is approximately two hours long, structured into 3 sections and 18 lessons, and targets associate-level learners with basic Databricks and MLflow experience.

## Key claims
- Databricks Academy offers the Machine Learning Model Development course for free.
- The course is the second in the Machine Learning with Databricks learning series.
- MLflow is used to log runs, track model performance, and manage model development from experimentation to iteration.
- Optuna is used for hyperparameter tuning to improve model performance in a structured and efficient way.
- Databricks AutoML can be accessed through both the UI and API to run experiments and compare models.
- The course consists of 3 sections and 18 lessons with a total duration of approximately 2 hours.
- Prerequisites include familiarity with Python, Spark, Delta Lake, Unity Catalog, and feature engineering fundamentals.
