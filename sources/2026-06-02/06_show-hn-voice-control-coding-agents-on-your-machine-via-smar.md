## Summary
The author developed a voice-controlled coding agent application that allows users to interact with their machines via a smartwatch, CarPlay, or phone. Frustrated with desk-bound coding and the limitations of phone keyboards, the app enables users to dispatch coding tasks and receive results through voice commands, leveraging SSH sessions to their own machines. The application aims to increase productivity by allowing coding "on the go" and offers both cloud-hosted and self-hosted backend options, prioritizing user privacy and cost-effectiveness.

## Key claims
- The application enables voice-controlled coding agents accessible via smartwatch, CarPlay, and phone.
- Users can interact with their own machines remotely through SSH sessions initiated by voice commands.
- The app interprets voice input, dispatches work to coding agents, and reads results back to the user.
- It offers context compression and message queuing for continuous interaction during active sessions.
- The application provides both cloud-hosted and self-hosted backend options, with self-hosting enhancing privacy.
- The app is free to use, with minimal costs incurred by the user for their existing machine and LLM API keys.
- The primary goal is to allow users to code remotely and "touch grass" without sacrificing productivity.
