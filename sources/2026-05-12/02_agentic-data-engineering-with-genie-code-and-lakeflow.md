## Summary
Databricks is integrating Genie Code more deeply into its Lakeflow platform to serve as an AI assistant for data engineers. The integration enables users to build Spark Declarative Pipelines from natural language prompts, orchestrate jobs, and manage scheduling without manual configuration. Genie Code can also debug pipelines by inspecting failures, proposing fixes, and showing diffs before changes are applied. Teams can further customize Genie Code with instructions, skills, and MCP connections to incorporate domain knowledge and external tools.

## Key claims
- Genie Code can generate and update Spark Declarative Pipelines from natural language prompts, including ingestion, transformations, and data quality checks.
- Genie Code integrates with Lakeflow Jobs to help define tasks, dependencies, scheduling, and monitoring.
- Genie Code supports CDC patterns, Auto Loader, expectations, and Declarative Automation Bundles for CI/CD-friendly projects.
- Genie Code can inspect pipeline behavior, analyze failures, propose fixes, and show diffs before changes are applied.
- Teams can customize Genie Code with instructions, skills, and MCP connections to bring in domain knowledge and internal workflows.
