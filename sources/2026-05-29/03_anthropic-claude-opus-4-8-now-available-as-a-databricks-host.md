## Summary
Databricks Model Serving has added support for Anthropic Claude Opus 4.8 as a Databricks-hosted model. Users can access the model through Foundation Model APIs using pay-per-token pricing. The model supports reasoning and vision query capabilities. This release expands Databricks' hosted model offerings with Anthropic's latest Claude Opus iteration.

## Key claims
- Databricks Model Serving now supports Anthropic Claude Opus 4.8 as a hosted model.
- Claude Opus 4.8 is accessible via Foundation Model APIs on a pay-per-token basis.
- The model supports reasoning query capabilities.
- The model supports vision query capabilities.
- Claude Opus 4.8 is available on the AWS deployment of Databricks.
