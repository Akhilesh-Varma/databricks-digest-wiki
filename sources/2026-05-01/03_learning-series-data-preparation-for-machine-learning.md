## Summary
Databricks Academy offers a free course called 'Data Preparation for Machine Learning' aimed at associate-level data scientists and ML practitioners. The course is the first in the Machine Learning with Databricks series and covers exploring, cleaning, and engineering features for traditional machine learning on the Databricks Data Intelligence Platform. Key topics include using Spark for data profiling and cleaning, managing features with Feature Store and Lakebase, and leveraging tools like Genie Code Agent Mode and Serverless Compute. Recent updates added Genie Code Agent Mode integration, Serverless Compute compatibility, and expanded Lakebase coverage. The course is free, takes approximately two hours, and consists of 17 lessons across 3 sections.

## Key claims
- Databricks Academy offers a free 'Data Preparation for Machine Learning' course for associate-level data scientists and ML practitioners.
- The course is the first in the Machine Learning with Databricks learning series.
- Learners use Spark and visualizations to profile data, identify missing values or outliers, and apply cleaning steps.
- Feature Store and Lakebase are used to build, transform, and keep features consistent across training and inference.
- The course has been updated to integrate Genie Code Agent Mode for conversational exploratory analysis.
- Notebooks have been updated for full compatibility with Serverless Compute, removing dependencies on classic clusters.
- The course is free, approximately two hours long, and structured as 17 lessons across 3 sections.
