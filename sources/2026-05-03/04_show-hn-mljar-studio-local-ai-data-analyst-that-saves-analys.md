## Summary
MLJAR Studio is a desktop application built on top of the open-source mljar-supervised AutoML library that lets users analyze tabular data through natural language conversations. The AI generates and executes Python code locally, and the entire conversation is saved as a reproducible Jupyter notebook (.ipynb) file. It supports multiple operating systems, auto-installs missing packages, and connects to various databases and file formats. For AI inference it supports Ollama locally, OpenAI, or an MLJAR-specific add-on. The app is priced at a one-time fee of $199 with a 7-day trial.

## Key claims
- MLJAR Studio converts natural language data conversations into reproducible Jupyter notebooks (.ipynb files).
- The app sets up a local Python environment automatically and runs on Mac, Windows, and Linux.
- It includes built-in AutoML for tabular data supporting classification, regression, and multiclass tasks.
- Users can connect to PostgreSQL, MySQL, SQL Server, Snowflake, Databricks, and Supabase.
- AI inference can run fully locally via Ollama with zero data egress, or via OpenAI or an MLJAR add-on.
- MLJAR Studio is priced at a one-time payment of $199 with a 7-day free trial.
- The product is positioned between Jupyter Notebook (flexible but manual) and cloud AI tools that do not preserve reproducible workflows.
