## Summary
Databricks has introduced Agent Mode in Genie Spaces, a new capability that enables Genie to investigate data more like a human analyst. Rather than responding to a single query, Genie now plans, runs multiple queries, and tests hypotheses to answer complex business questions such as why churn spiked or how spending changes could impact results. The feature leverages Genie space semantics and Unity Catalog metadata to stay grounded in relevant business context. Agent Mode is integrated into both Genie Spaces and AI BI Dashboards and can be enabled once for use across both surfaces.

## Key claims
- Databricks introduced Agent Mode in Genie Spaces as a new agentic data investigation capability.
- Agent Mode allows Genie to plan, run multiple queries, and test hypotheses instead of returning a single query and chart.
- The feature supports 'why' and 'what if' questions, such as why churn spiked or how a change in spend could impact results.
- Agent Mode uses Genie space semantics and Unity Catalog metadata to focus on the most relevant data.
- The system applies right-sized reasoning, moving quickly on simple prompts and spending more time on complex questions.
- Agent Mode is built into both Genie Spaces and AI BI Dashboards and can be enabled once for use across both.
