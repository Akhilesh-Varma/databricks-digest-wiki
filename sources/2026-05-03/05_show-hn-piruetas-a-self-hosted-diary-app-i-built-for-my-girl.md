## Summary
A developer built Piruetas, a self-hosted diary web application, after failing to find a suitable journaling tool for his girlfriend. The app features a day-per-page layout with rich text editing, drag-and-drop image uploads, auto-save, and a clean mobile UI. It supports both personal and multi-user deployments via Docker Compose. After positive reception from his girlfriend and internal QA, the developer open-sourced the project on GitHub and made a live demo available.

## Key claims
- Piruetas is a self-hosted diary app built because existing solutions were too complex, too simple, or required trusting a third-party cloud service.
- The name 'Piruetas' means 'pirouettes' in Spanish and was chosen by the developer's girlfriend.
- The app supports rich text editing, drag-and-drop image uploads, auto-save, and public share links.
- Piruetas can be deployed for personal or multi-user use via Docker Compose.
- A live demo is available at piruet.app with data resetting every 30 minutes.
- The project is open-sourced on GitHub under the repository patillacode/piruetas.
