## Summary
Databricks has announced a Public Preview feature that allows external Delta and Iceberg clients to read streaming tables and materialized views managed by a pipeline. This access is facilitated through the Unity Catalog and Iceberg catalog REST APIs. The key benefit is that data does not need to be copied, enabling direct external access to pipeline-managed data.

## Key claims
- External Delta and Iceberg clients can now read streaming tables and materialized views managed by a Databricks pipeline.
- Access is provided through the Unity Catalog and Iceberg catalog REST APIs.
- The feature eliminates the need to copy data for external access.
- The feature is currently in Public Preview.
