## Summary
This paper proposes a unified secure-by-design generative AI framework combining PromptShield and the Cloud Investigation Automation Framework (CIAF) to address cybersecurity and forensic challenges in cloud environments. PromptShield defends large language models against prompt injection attacks using ontology-driven input validation, while CIAF automates cloud forensic investigations across all six phases of the forensic process using structured ontology-based reasoning. The system is evaluated on real-world datasets from AWS and Microsoft Azure, achieving precision, recall, and F1 scores above 93% under adversarial conditions. CIAF also improves ransomware detection accuracy in cloud logs using Likert-transformed performance features. Together, the framework advances automation, interpretability, and trustworthiness for AI-driven incident response at scale.

## Key claims
- PromptShield uses ontology-driven validation to defend LLMs against prompt injection attacks, achieving precision, recall, and F1 scores above 93% under attack conditions.
- CIAF automates cloud forensic investigations through structured ontology-based reasoning across all six phases of the forensic process.
- The integrated framework was evaluated on real-world datasets from AWS and Microsoft Azure.
- CIAF enhances ransomware detection accuracy in cloud logs using Likert-transformed performance features.
- The framework addresses both LLM security vulnerabilities and the lack of forensic rigor in existing LLM-based systems.
- The proposed system offers a scalable foundation for real-time, AI-driven incident response across diverse cloud infrastructures.
