## Summary
The author, a 10-year AI-in-finance veteran, observed that many startups lack the data science background needed to build systematic evaluation systems for AI agents. To address this gap, they condensed their experience into a Claude Skill—a practical tool that automatically sets up baseline evaluation code directly in a user's codebase. The tool is designed so that a user simply tells Claude they need evals, and it generates evaluation patterns based on proven practices. The resulting evals produce a summary of agent strengths and weaknesses, serving as a starting point for teams without dedicated evaluation expertise.

## Key claims
- The author has 10 years of experience working on AI in finance, with a focus on evaluation systems for production environments.
- Many startup teams building AI agents lack fluency in systematic evaluation processes.
- Large organizations rarely face this gap due to dedicated evaluation teams, but startups without data science backgrounds struggle.
- The author created a Claude Skill that sets up baseline agent evaluations directly in a user's codebase.
- Users trigger the tool by telling Claude they need evals, requiring no manual configuration.
- The generated evals follow patterns observed across many production environments and summarize agent performance strengths and weaknesses.
