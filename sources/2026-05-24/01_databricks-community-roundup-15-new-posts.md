## Summary
This Databricks community roundup highlights 15 recent technical blog posts covering a wide range of topics on the Databricks platform. Key themes include query performance tuning for Serverless SQL, AI BI dashboard optimization, and Unity Catalog governance enhancements such as ABAC data classification now reaching general availability. The roundup also covers new beta features like Unity AI Gateway with LLM guardrails and cost controls, Databricks Container Services for standard compute, and support for importing Tableau and Power BI files into Genie Code. Additional posts address data experimentation best practices, materialized view refresh policies, stateful streaming testing, and solution accelerators for media mix modeling and compliance.

## Key claims
- Databricks Serverless SQL (DBSQL) is positioned as the latest offering for building data warehouses on the Lakehouse.
- Databricks has expanded Unity AI Gateway in beta with LLM guardrails, payload logging, cost controls, and MCP governance capabilities.
- Unity Catalog's ABAC row filtering, column masking, governed tags, and data classification features have reached general availability.
- Genie Code now supports importing Tableau and Power BI files to automatically build AI BI dashboards connected to metric views.
- Databricks Container Services for standard compute with custom Docker images in shared compute is available in beta.
- Databricks Academy offers a free 'Master Modern Analytics Fundamentals' training course on the Databricks Platform.
- A REFRESH POLICY feature for materialized views aims to provide more predictable billing by preventing unexpected full refreshes.
