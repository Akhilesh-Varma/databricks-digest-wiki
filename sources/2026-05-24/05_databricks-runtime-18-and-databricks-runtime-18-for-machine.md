## Summary
Databricks has released Databricks Runtime 18 and Databricks Runtime 18 for Machine Learning in Beta, both powered by Apache Spark 4.1.0. This release introduces a unified release notes format, consolidating what were previously separate pages for each feature version (18.0, 18.1, 18.2) into a single versioned page. Going forward, features that would have shipped as version 18.3 or later will instead appear as dated updates to the Databricks Runtime 18 release notes page.

## Key claims
- Databricks Runtime 18 is now in Beta.
- Databricks Runtime 18 for Machine Learning is now in Beta.
- Both runtimes are powered by Apache Spark 4.1.0.
- Databricks Runtime 18 is the first release to use a unified release notes format.
- Previously, each feature version (18.0, 18.1, 18.2) had its own release notes page.
- Features that would have shipped as 18.3 or later will now ship as dated updates to Databricks Runtime 18.
