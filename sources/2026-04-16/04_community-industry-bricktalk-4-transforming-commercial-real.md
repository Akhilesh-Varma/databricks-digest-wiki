## Summary
Databricks is hosting a virtual BrickTalk session on April 16 focused on transforming commercial real estate portfolio management using AI. The session will demonstrate a Lease Extraction Platform built entirely on Databricks that converts complex lease PDFs into structured, risk-aware portfolio insights. The platform uses AI agents, model serving endpoints, Unity Catalog with medallion architecture, and a full-stack React/Flask application. Key capabilities include tenant credit risk scoring, financial enrichment via MCP web search, and portfolio analytics dashboards. The event targets finance professionals and showcases real-world architectures and demos.

## Key claims
- The Lease Extraction Platform can extract key lease data points from PDFs within seconds using Databricks AI capabilities.
- The platform automatically assesses tenant credit risk, concentration risk, and rollover exposure.
- Financial enrichment of landlord and tenant profiles is performed via real-time MCP web search.
- The solution is built entirely on Databricks using Unity Catalog with a bronze-silver-gold medallion architecture.
- Databricks Asset Bundles enable one-command deployment across development, staging, and production environments.
- The BrickTalk event was rescheduled to April 16 at 9 AM PT, with registrations automatically updated.
