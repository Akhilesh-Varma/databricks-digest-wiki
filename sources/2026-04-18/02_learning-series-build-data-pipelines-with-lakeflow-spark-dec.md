## Summary
Databricks Academy offers a course on building data pipelines with Lakeflow Spark Declarative Pipelines. The course covers core concepts, including building incremental batch and streaming pipelines, configuring compute and data assets, and monitoring pipelines. It is designed for data engineers and learners with Databricks basics. The course is free and consists of 3 sections and 22 lessons. It aims to teach students how to build end-to-end pipelines using Lakeflow Spark Declarative Pipelines.
