## Summary
SiriusHelper is a deployed LLM agent-based intelligent assistant designed to support operations on big data platforms. It automatically identifies user intent and routes queries to specialized expert workflows, such as SQL execution diagnosis. The system uses a DeepSearch-driven multi-hop retrieval mechanism combined with a priority-based hierarchical knowledge base to improve answer reliability and reduce latency. It also features automated ticket understanding and SOP distillation to continuously enrich its knowledge base and reduce expert overhead. Deployed on Tencent's Big Data platform, SiriusHelper reduced online ticket volume by 20.8%.

## Key claims
- SiriusHelper is a deployed intelligent assistant for big data platforms that automatically identifies user intent and routes queries to appropriate handling paths.
- The system combines a DeepSearch-driven mechanism with a priority-based hierarchical knowledge base to enable multi-hop retrieval without context overload.
- SiriusHelper introduces automated ticket understanding and SOP distillation to diagnose assistant failures and extract domain-specific SOPs.
- Deployment on Tencent's Big Data platform resulted in a 20.8% reduction in online ticket volume.
- Existing LLM RAG assistants face challenges including limited scenario coverage, inefficient multi-hop retrieval, and high maintenance costs from unstructured escalated tickets.
- SiriusHelper outperforms representative alternative systems in experiments and online deployment evaluations.
