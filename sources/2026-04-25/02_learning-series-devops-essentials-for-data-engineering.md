## Summary
Databricks Academy has launched a free course, "DevOps Essentials for Data Engineering," aimed at data engineers. The course emphasizes applying software engineering best practices and DevOps principles to the Databricks Data Intelligence Platform. It focuses on core habits for building, testing, and maintaining data pipelines, covering topics like version control, CI/CD, unit testing with pytest, integration testing, and deployment options.

## Key claims
- Databricks Academy offers a free course on DevOps Essentials for Data Engineering.
- The course teaches data engineers to apply software engineering best practices and DevOps principles on the Databricks platform.
- Key topics include code quality, version control, testing, CI/CD, modularity in PySpark, and unit/integration testing.
- The course covers using Git operations in Databricks for continuous integration.
- It compares various deployment options for Databricks assets, including REST API, CLI, and SDK.
- The course is designed for experienced Databricks users comfortable with PySpark, SQL, Python, and Git.
