## Summary
Databricks has released the public preview of Lakeflow Designer, a no-code, AI-native tool for data preparation and analysis. It allows analysts and business users to build workflows visually within the Databricks environment, leveraging Unity Catalog for governance and permissions. The tool features AI-assisted authoring with Genie, step-by-step reviewable transformations, and generates production-ready Python code for integration into existing pipelines.

## Key claims
- Databricks has launched the public preview of Lakeflow Designer.
- Lakeflow Designer offers a visual, no-code, AI-native approach to data preparation and analysis.
- Users can build workflows using a drag-and-drop canvas and natural language, avoiding the need to write SQL or Python.
- The tool operates directly on native lakehouse data, integrating with Unity Catalog for governance, lineage, and permissions.
- Genie, Databricks' AI coding assistant, can generate or modify workflows based on plain English descriptions.
- Each transformation in Lakeflow Designer is a visual operator with data previews, enhancing inspectability and trust.
- Visual flows generate Python code that can be versioned, scheduled with Lakeflow Jobs, and integrated into production pipelines.
