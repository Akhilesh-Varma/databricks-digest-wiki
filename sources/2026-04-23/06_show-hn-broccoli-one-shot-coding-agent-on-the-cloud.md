## Summary
The authors built Broccoli, an open-source coding agent that runs tasks in isolated cloud sandboxes and opens PRs for human review. It was created to solve the problem of managing multiple coding tasks at once. Broccoli checks out the repo, works through an implementation, runs tests and review loops, and opens a PR for inspection. Over four weeks, 100 PRs from non-developers were shipped via Broccoli, with a 60% share for developers. The implementation uses various tools such as Webhook deployment, Sandbox, Project management, and Code hosting.

## Key claims
- Broccoli is an open-source harness for taking coding tasks and running them in isolated cloud sandboxes.
- Broccoli opens PRs for human review after executing tasks end to end independently.
- 100 PRs from non-developers were shipped via Broccoli over four weeks.
- Developers on the team had a 60% share of PRs shipped via Broccoli.
- Broccoli uses tools like Webhook deployment, Sandbox, and Project management.
