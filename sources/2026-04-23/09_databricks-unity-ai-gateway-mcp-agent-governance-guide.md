## Summary
Databricks has introduced Unity AI Gateway to address the issue of agent sprawl in enterprise AI adoption. The gateway provides a unified way to track and govern AI-powered applications, ensuring proper audit trails and access control. It extends the Unity Catalog permission model to cover LLM endpoints, MCP servers, and coding agents. The gateway acts as a proxy layer between agents and external systems, evaluating requests against access policies and logging them to a centralized audit table. This solution aims to solve the governance gap created by the shift to agentic AI workflows.
