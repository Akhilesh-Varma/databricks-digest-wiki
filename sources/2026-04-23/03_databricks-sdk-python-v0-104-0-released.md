## Summary
Databricks has released version 0.104.0 of its Python SDK, which includes bug fixes and internal changes. The update adds the X-Databricks-Org-Id header to WorkspaceExt.upload and WorkspaceExt.download for SPOG host compatibility. It also expands AI agent detection to include Goose, Amp, Augment, Copilot VS Code, Kiro, and Windsurf. The release honors the AGENT standard and resolves to a known product if the value matches one. The update also includes presence-only env var matchers that treat an empty string as set for parity with the Go and Java SDKs.
